# ***UML 用例图关系（极简速记）

# 一、真题

A 扩展关系
B 聚合关系
C 包含关系
D 继承关系

------

# 二、答案

```text
B 聚合关系
```

------

# 三、用例之间的三种关系（超高频）

```text
1. 包含（include）
2. 扩展（extend）
3. 泛化/继承（generalization）
```


```text

```

