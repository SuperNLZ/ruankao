# 关系代数：自然连接等价转换（详细解题）

# 一、题目

给定关系：

```text
R(A, B, C, D)
S(A, C, E, F)
```

以下哪一个表达式与：

```text
σ R.B > S.E (R ⋈ S)
```

等价？

正确答案：

```text
B
```

即：

```text
π 1,2,3,4,7,8(
  σ 1=5 ∧ 2>7 ∧ 3=6 (R×S)
)
```

---

# 二、涉及的关系代数符号

## 1. σ：选择运算

σ 读作：

```text
选择
```

作用：

```text
选行
```

例如：

```text
σ B > 10(R)
```

意思是：

```text
从 R 中选出 B > 10 的那些行
```

---

## 2. π：投影运算

π 读作：

```text
投影
```

作用：

```text
选列
```

例如：

```text
π A,B(R)
```

意思是：

```text
只保留 R 中的 A、B 两列
```

---

## 3. ×：笛卡尔积

× 表示：

```text
两个关系的所有行两两组合
```

结果列数：

```text
R 的列数 + S 的列数
```

---

## 4. ⋈：自然连接

⋈ 表示：

```text
自然连接
```

自然连接会：

```text
自动按同名属性相等连接
并去掉重复列
```

---

# 三、先理解原式

原式：

```text
σ R.B > S.E (R ⋈ S)
```

意思是：

```text
先对 R 和 S 做自然连接
再选出满足 R.B > S.E 的记录
```

也就是：

```text
自然连接 + 条件筛选
```

---

# 四、R 和 S 的同名属性

关系：

```text
R(A, B, C, D)
S(A, C, E, F)
```

同名属性是：

```text
A
C
```

所以自然连接：

```text
R ⋈ S
```

要求：

```text
R.A = S.A
R.C = S.C
```

---

# 五、把自然连接转换成笛卡尔积 + 选择 + 投影

自然连接的标准转换公式：

```text
R ⋈ S
=
π 去掉重复列 (
  σ 同名属性相等 (R × S)
)
```

本题中就是：

```text
R ⋈ S
=
π 1,2,3,4,7,8(
  σ 1=5 ∧ 3=6 (R×S)
)
```

然后题目外面还有条件：

```text
R.B > S.E
```

所以再加上：

```text
2 > 7
```

最终：

```text
π 1,2,3,4,7,8(
  σ 1=5 ∧ 2>7 ∧ 3=6 (R×S)
)
```

---

# 六、为什么是这些列号

先做：

```text
R × S
```

属性顺序默认是：

```text
R 的所有属性 + S 的所有属性
```

所以：

| 列号 | 属性 |
|---:|---|
| 1 | R.A |
| 2 | R.B |
| 3 | R.C |
| 4 | R.D |
| 5 | S.A |
| 6 | S.C |
| 7 | S.E |
| 8 | S.F |

---

# 七、选择条件怎么来的

## 1. 自然连接条件 A 相等

因为 R 和 S 都有 A：

```text
R.A = S.A
```

对应列号：

```text
1 = 5
```

---

## 2. 自然连接条件 C 相等

因为 R 和 S 都有 C：

```text
R.C = S.C
```

对应列号：

```text
3 = 6
```

---

## 3. 题目筛选条件

题目要求：

```text
R.B > S.E
```

对应列号：

```text
2 > 7
```

注意：

```text
7 表示第 7 列 S.E
不是数字常量 7
```

所以不能写成：

```text
2 > '7'
```

---

# 八、σ 条件整体含义

```text
σ 1=5 ∧ 2>7 ∧ 3=6 (R×S)
```

意思是：

```text
从 R×S 的所有组合中选出满足以下条件的行：

1. 第1列 = 第5列
2. 第2列 > 第7列
3. 第3列 = 第6列
```

对应实际属性就是：

```text
R.A = S.A
R.B > S.E
R.C = S.C
```

---

# 九、为什么要用 π 1,2,3,4,7,8

自然连接会去掉重复属性。

R 和 S 同名属性：

```text
A
C
```

在 R×S 中：

```text
R.A 是第1列
S.A 是第5列
R.C 是第3列
S.C 是第6列
```

因为自然连接已经要求：

```text
1 = 5
3 = 6
```

所以：

```text
第5列 S.A 是重复列
第6列 S.C 是重复列
```

要删除第 5、6 列，保留：

```text
1,2,3,4,7,8
```

即：

```text
A, B, C, D, E, F
```

所以使用：

```text
π 1,2,3,4,7,8
```

# 十、完整等价过程

原式：

```text
σ R.B > S.E (R ⋈ S)
```

第一步：把自然连接换成笛卡尔积加条件：

```text
R ⋈ S
=
π 1,2,3,4,7,8(
  σ 1=5 ∧ 3=6 (R×S)
)
```

第二步：加入题目额外筛选条件：

```text
R.B > S.E
```

对应：

```text
2 > 7
```

最终：

```text
π 1,2,3,4,7,8(
  σ 1=5 ∧ 2>7 ∧ 3=6 (R×S)
)
```

# 十一、为什么选 B

选项 B：

```text
π 1,2,3,4,7,8(
  σ 1=5 ∧ 2 > 7 ∧ 3=6(R×S)
)
```

完全对应：

```text
R.A = S.A
R.B > S.E
R.C = S.C
去掉重复列 S.A、S.C
```

所以选：

```text
B
```

---

# 十二、为什么其他选项错

## A 错

```text
σ 2 > 7(R×S)
```

问题：

```text
只做了 R.B > S.E
没有做自然连接条件 R.A=S.A 和 R.C=S.C
也没有去掉重复列
```

---

## C 错

```text
σ 2 > '7'(R×S)
```

问题：

```text
'7' 是字符常量
不是第7列
```

而题目要比较的是：

```text
第2列 R.B > 第7列 S.E
```

---

## D 错

```text
2 > '7'
```

同样错误：

```text
'7' 是字符常量
不是第7列 S.E
```

