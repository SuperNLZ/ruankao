# 数据链路层功能（软考快速记忆）

## 题目

在计算机网络的分层体系结构中，数据链路层负责在相邻节点之间提供可靠的数据传输服务。下列选项中，不属于数据链路层功能的是（）。

A. 封装成帧  
B. 路由选择  
C. 错误检测  
D. 透明传输

---

# 正确答案

## B：路由选择

---

# 一、为什么选 B

## 路由选择属于：

```text
网络层
```

而不是：

```text
数据链路层
```

---

# 二、数据链路层的核心任务

数据链路层：

```text
负责相邻节点之间的数据传输
```

重点：

```text
点到点
```

不是：

```text
跨网络路径选择
```

---

# 三、数据链路层的主要功能（高频）

| 功能 | 说明 |
|---|---|
| 封装成帧 | 把比特流组织成帧 |
| 差错检测 | 检测传输错误 |
| 透明传输 | 解决控制字符冲突 |
| 流量控制 | 控制发送速度 |
| MAC寻址 | 使用MAC地址通信 |

---

# 四、四个选项分析

## A 封装成帧

属于数据链路层。

作用：

```text
把数据组织成“帧”
```

因此：

```text
正确功能
```

---

## B 路由选择

属于：

```text
网络层
```

例如：

- IP路由
- 最短路径
- RIP
- OSPF

都属于网络层。

因此：

```text
不是数据链路层功能
```

---

## C 错误检测

属于数据链路层。

常见：

```text
CRC循环冗余校验
```

用于检测：

```text
帧是否出错
```

---

## D 透明传输

属于数据链路层。

作用：

```text
避免数据中的控制字符
被误认为控制信息
```

例如：

- 字节填充
- 比特填充

---

# 五、什么是透明传输（高频易错）

透明传输：

```text
无论数据是什么
都能正常传输
```

例如：

数据里恰好出现：

```text
帧结束标志
```

系统也不能误判。

因此需要：

```text
转义/填充
```

---

# 六、数据链路层 vs 网络层（非常重要）

| 层次 | 核心任务 |
|---|---|
| 数据链路层 | 相邻节点通信 |
| 网络层 | 路由与路径选择 |

---

# 七、软考快速记忆口诀

## 数据链路层

```text
成帧
检错
透明
```

---

## 网络层

```text
找路
```

即：

```text
路由选择
```

---

# 八、OSI 各层高频功能（超高频）

| 层 | 典型功能 |
|---|---|
| 物理层 | 比特传输 |
| 数据链路层 | 成帧、检错 |
| 网络层 | 路由选择 |
| 传输层 | 端到端通信 |
| 应用层 | 用户服务 |

---

# 九、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 路由属于链路层 | 错，属于网络层 |
| CRC属于网络层 | 错，属于数据链路层 |
| 透明传输属于物理层 | 错，属于数据链路层 |
| MAC地址属于网络层 | 错，属于数据链路层 |

---

# 十、一句话总结

> 数据链路层负责相邻节点之间的数据传输，其核心功能包括封装成帧、差错检测和透明传输；而路由选择属于网络层功能。
