# EDI（Electronic Data Interchange）电子数据交换（软考快速记忆）

## 一、什么是 EDI

EDI：

```text
Electronic Data Interchange
电子数据交换
```

是：

```text
企业之间
按照统一标准
通过计算机网络
自动交换商业数据
```

的一种技术。

---

# 二、EDI 的核心目标

实现：

```text
无纸化办公
自动化数据交换
```

常用于：

- 订单
- 发票
- 物流
- 海关
- 银行
- 供应链

---

# 三、EDI 的四个基本模块（高频）

| 模块 | 作用 |
|---|---|
| 用户接口模块 | 提供人与EDI系统交互 |
| 内部接口模块 | 企业内部系统数据转换 |
| 报文生成与处理模块 | EDI标准报文处理 |
| 通信模块 | 网络传输EDI报文 |

---

# 四、四个模块详解

## 1. 用户接口模块

负责：

```text
用户操作
输入输出
界面交互
```

例如：

- 查询订单
- 查看报文
- 操作EDI系统

---

## 2. 内部接口模块

负责：

```text
企业内部数据
与EDI格式转换
```

例如：

```text
ERP 数据
→
EDI标准格式
```

或者：

```text
EDI报文
→
企业数据库格式
```

---

## 3. 报文生成与处理模块

负责：

```text
EDI标准报文
生成、翻译、校验
```

核心：

```text
标准化
```

例如：

- ANSI X12
- EDIFACT

---

## 4. 通信模块

负责：

```text
EDI数据传输
```

通过：

- Internet
- 专用网络
- VAN

进行通信。

---

# 五、EDI 工作流程（非常重要）

```text
企业数据
   ↓
内部接口模块
   ↓
报文生成与处理模块
   ↓
通信模块
   ↓
另一企业
```

用户通过：

```text
用户接口模块
```

进行操作。

---

# 六、软考高频关键词

EDI 特点：

```text
标准化
结构化
自动化
无纸化
```

---

# 七、四个模块快速记忆口诀

## 口诀

```text
人接内报通
```

对应：

| 口诀 | 模块 |
|---|---|
| 人 | 用户接口模块 |
| 接 | 内部接口模块 |
| 报 | 报文生成与处理模块 |
| 通 | 通信模块 |

---

# 八、高频易错点

| 易错点 | 正确理解 |
|---|---|
| EDI 是邮件系统 | 错，EDI 是标准化商业数据交换 |
| EDI 不需要标准 | 错，EDI 强调统一标准 |
| EDI 是人工录入 | 错，强调自动交换 |
| 通信模块负责业务处理 | 错，只负责传输 |

---

# 九、软考标准表述

> EDI 系统通常由用户接口模块、内部接口模块、报文生成与处理模块以及通信模块四部分组成，用于实现企业之间标准化、自动化的数据交换。

---

# 十、一句话总结

> EDI 的核心是：按照统一标准，通过网络自动交换商业数据，其四个基本模块包括用户接口、内部接口、报文处理和通信模块。
