# 进程通信体系结构（软考快速记忆）

# 一、什么是进程通信（IPC）

IPC：

```text
Inter-Process Communication
进程间通信
```

表示：

```text
不同进程之间
交换数据与同步协作
```

的机制。

---

# 二、为什么需要进程通信

因为：

```text
进程之间相互独立
```

每个进程：

- 有自己的地址空间
- 默认不能直接访问彼此数据

因此需要：

```text
IPC机制
```

实现：

- 数据交换
- 同步控制
- 协同工作

---

# 三、进程通信体系结构分类（高频）

进程通信主要分为：

| 类型 | 特点 |
|---|---|
| 共享存储 | 共享同一块内存 |
| 消息传递 | 通过消息交换 |
| 管道通信 | 文件流方式通信 |
| 信号量机制 | 实现同步互斥 |
| Socket通信 | 网络进程通信 |

---

# 四、共享存储（Shared Memory）

## 核心思想

```text
多个进程
共享同一块内存
```

例如：

```text
进程A 写数据
进程B 直接读取
```

---

## 特点

### 优点

```text
速度最快
```

因为：

```text
无需内核频繁拷贝
```

---

### 缺点

```text
需要同步控制
```

否则：

```text
可能同时修改数据
```

---

## 常配合

```text
信号量
```

使用。

---

# 五、消息传递（Message Passing）

## 核心思想

```text
进程通过发送消息通信
```

例如：

```text
send(message)
receive(message)
```

---

## 特点

### 优点

```text
解耦
安全
```

进程之间：

```text
无需共享内存
```

---

### 缺点

```text
效率低于共享内存
```

因为：

```text
需要消息复制
```

---

## 典型应用

- 微服务
- MQ消息队列
- 分布式系统

---

# 六、管道通信（Pipe）

## 核心思想

```text
像水管一样
按字节流传输
```

---

## 特点

### 半双工

```text
数据单向流动
```

---

## 分类

| 类型 | 特点 |
|---|---|
| 匿名管道 | 父子进程通信 |
| 命名管道FIFO | 无亲缘关系进程 |

---

## Linux 示例

```bash
ls | grep txt
```

其中：

```text
|
```

就是管道。

---

# 七、信号量机制（Semaphore）

## 核心作用

```text
同步
互斥
```

不是主要用于传数据。

---

## 常见操作

| 操作 | 含义 |
|---|---|
| P操作 | 申请资源 |
| V操作 | 释放资源 |

---

## 用途

解决：

- 临界区
- 生产者消费者
- 进程同步

---

# 八、Socket通信（超高频）

## 核心思想

```text
网络中的进程通信
```

支持：

- 本机
- 跨机器
- Internet

---

## 特点

```text
最通用
```

现代：

- Web
- 微服务
- RPC
- 分布式系统

基本都基于 Socket。

---

# 九、进程通信方式对比（超高频）

| 通信方式 | 速度 | 是否跨机器 | 是否共享内存 |
|---|---|---|---|
| 共享内存 | 快 | 否 | 是 |
| 消息传递 | 中 | 可支持 | 否 |
| 管道 | 中 | 否 | 否 |
| 信号量 | 不传数据 | 否 | 否 |
| Socket | 较慢 | 是 | 否 |

---

# 十、同步与互斥（高频）

## 互斥

```text
同一时刻
只能一个进程访问资源
```

例如：

```text
打印机
```

---

## 同步

```text
控制执行顺序
```

例如：

```text
先生产
后消费
```

# 十二、高频易错点

| 易错点 | 正确理解 |
|---|---|
| 信号量用于大量数据传输 | 错，主要用于同步互斥 |
| 管道支持双向通信 | 一般半双工 |
| 共享内存不需要同步 | 错，必须同步 |
| Socket只能跨网络 | 本机也可使用 |

---

# 十三、典型考试对应关系

| 需求 | 最适合方式 |
|---|---|
| 速度最快 | 共享内存 |
| 网络通信 | Socket |
| 同步互斥 | 信号量 |
| 父子进程简单通信 | 管道 |
| 松耦合通信 | 消息传递 |
