# 设计模式分类判断解题过程

## 题目

关于设计模式的描述，正确的是（ ）。

```text
A. 装饰器模式属于行为模式
B. 原型模式属于创建型模式
C. 解释器和代理模式是同一类模式
D. 观察者模式属于结构型模式
```

## 答案

```text
B
```

## 解题过程

设计模式通常按目的分为三大类：

```text
创建型模式
结构型模式
行为型模式
```

逐项判断：

| 选项 | 判断 | 原因 |
|---|---|---|
| A | 错 | 装饰器模式属于结构型模式，不是行为型模式 |
| B | 对 | 原型模式属于创建型模式 |
| C | 错 | 解释器属于行为型模式，代理属于结构型模式 |
| D | 错 | 观察者模式属于行为型模式，不是结构型模式 |

因此正确答案是：

```text
B. 原型模式属于创建型模式
```

## 三类设计模式速记

### 创建型模式

关注：

```text
对象怎么创建
```

常见模式：

```text
单例
工厂方法
抽象工厂
建造者
原型
```

口诀：

```text
单工抽建原
```

### 结构型模式

关注：

```text
类和对象怎么组合
```

常见模式：

```text
适配器
桥接
组合
装饰器
外观
享元
代理
```

口诀：

```text
适桥组装外享代
```

### 行为型模式

关注：

```text
对象之间怎么协作、职责怎么分配
```

常见模式：

```text
责任链
命令
解释器
迭代器
中介者
备忘录
观察者
状态
策略
模板方法
访问者
```

口诀：

```text
责命解迭中备观状策模访
```

## 本题易错点

```text
装饰器 = 结构型
代理 = 结构型
解释器 = 行为型
观察者 = 行为型
原型 = 创建型
```

## 最终背诵版

```text
创建型：单工抽建原。
结构型：适桥组装外享代。
行为型：责命解迭中备观状策模访。
本题中只有“原型模式属于创建型模式”正确，所以选 B。
```
